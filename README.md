# Django REST Framework with Simple JWT

- Custom user model
- Nested routing with [drf-nested-routers](https://github.com/alanjds/drf-nested-routers)
- User can only have 1 reaction to a Tweet

# Getting started

- create a `.env` file and add `SECRET`
